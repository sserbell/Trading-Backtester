# Trading-Backtester
A Python backtester for stock strategies using moving averages."

---

## Features
- Fetch historical stock data from Yahoo Finance
- Calculate long and short moving averages
- Generate buy/sell signals based on crossover strategy
- Simulate trades and track portfolio value
- Visualize portfolio performance over time

---

## Tech Stack 
- **Language:** Python
- **Data Analysis:** Pandas
- **Financial Data:** Yahoo Finance API (yfinance)
- **Backtesting Logic:** Custom Python code
- **Visualization:** Matplotlib

---

### How to Run
1. Install dependencies
2. Run the program (python main.py)
3. Enter the ticker, start date, and end date when prompted
